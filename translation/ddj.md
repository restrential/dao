# Dao De Jing — Translation

A translation of the Dao De Jing (道德經) into modern English.

## Goals
- Produce a faithful yet accessible English rendering of the DDJ
- Apply the terminology and style conventions from the broader translation project
- Preserve poetic rhythm and philosophical depth where possible

## Files

| File | Description |
|---|---|
| `jxz-english.md` | Jiang Xizhang (江希張) 1919 annotation + English translation. Chapters 1–10 complete. Published at https://books.tongde.us/books/ddj-jxz. Renamed from `jxz.md`. |
| `jxz-chinese-s.md` | Full original Chinese text (Simplified), all 81 chapters. Imported from `新注道德经白话解说——江希张.doc`. Source OCR: `inputs/processed/新注道德经白话解说——江希张.txt`. Renamed from `jxz-chinese.md`. |
| `jxz-chinese-t.md` | Traditional Chinese (Taiwan standard) version of `jxz-chinese-s.md`. Generated via `opencc -c s2tw.json`. Note: ch. 6 has a known conversion error (`穀` should be `谷`). |
| `jxz-chinese-t-punct.md` | Copy of `jxz-chinese-t.md` with all 81 章 `原文` sections re-punctuated using standard modern Chinese punctuation (，。；：？！). Preview texts at end of 演說 sections left unpunctuated. |
| `jxz-chinese-discrepancies.md` | Systematic comparison of JXZ's 原文 against the Wang Bi edition (王弼本), all 81 chapters. Documents orthographic variants, intentional textual variants, possible scribal/OCR errors, and cross-file issues. |
